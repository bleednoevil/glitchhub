# 🔥 GlitchHub

Transform GitHub’s global event stream into a playable MIDI interface.

Each push event lights up a pad. Press a pad to open the repository.

## 🎛 Features
- Real-time GitHub public events
- MIDI grid visualization (Push / Launchpad)
- Pad decay animation
- Deterministic repo-to-pad mapping
- Press pad to open repository in browser

## 🧠 Concept

GlitchHub turns GitHub into a real-time signal source.

- Repositories appear as pulses of light
- Activity fades over time
- Interaction happens through a MIDI controller
